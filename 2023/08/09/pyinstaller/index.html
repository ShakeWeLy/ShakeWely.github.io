程序结构图

  • 初始化界面

    • 创建主窗口
    • 添加标签和输入框
    • 添加按钮 “新建文章”, “生成”, “部署”, “初始化Hexo目录”, “选择Hexo目录”
    • 创建配置文件 “config.ini”

  • 主循环

    • 用户操作界面按钮
      • “新建文章” 按钮
        • 输入文章标题
        • 读取配置文件中的 Hexo 目录地址
        • 执行 “hexo n” 命令
        • 显示命令执行结果
      • “生成” 按钮
        • 读取配置文件中的 Hexo 目录地址
        • 执行 “hexo g” 命令
        • 显示命令执行结果
      • “部署” 按钮
        • 读取配置文件中的 Hexo 目录地址
        • 执行 “hexo d” 命令
        • 显示命令执行结果
      • “初始化Hexo目录” 按钮
        • 弹出文件选择对话框，选择 Hexo 目录
        • 保存选择的地址到配置文件
        • 显示初始化成功提示
      • “选择Hexo目录” 按钮
        • 弹出文件选择对话框，选择 Hexo 目录
        • 保存选择的地址到配置文件
        • 显示选择成功提示
