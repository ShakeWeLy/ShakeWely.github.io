2024年1月23日 python库总结

httpx

httpx 是一个比 requests 更适合现代 Python 应用（尤其是异步应用）的 HTTP 客户端。

特性 requests httpx
同步请求 ✅ ✅
异步请求 ❌ ✅
HTTP/2 支持 ❌ ✅
典型场景 脚本、爬虫、小型服务 现代 Web 服务、异步微服务、FastAPI、HTTP/2

Pytest

心目的只有一个：验证程序的实际运行结果是否符合你预期的正确值。

作用

🗝️什么时候必须用工具

如果你的项目：

  • 测试文件很多（几十几百个）
  • 每次手动执行麻烦
  • 需要在 CI/CD 里自动跑（比如提交 PR 时自动跑所有测试）
  • 需要统计覆盖率（哪些代码没被测到）
  • 需要做参数化、大量组合测试
