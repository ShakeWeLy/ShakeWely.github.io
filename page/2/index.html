python之项目总结

0 源码包与二进制包什么区别？

Python 包的分发可以分为两种：

1、以源码包的方式发布

源码包安装的过程，是先解压，再编译，最后才安装，所以它是跨平台的，由于每次安装都要进行编译，相对二进包安装方式来说安装速度较慢。

2、以二进制包形式发布

二进制包的安装过程省去了编译的过程，直接进行解压安装，所以安装速度较源码包来说更快。

由于不同平台的编译出来的包无法通用，所以在发布时，需事先编译好多个平台的包。

1 Python 中的 requirements.txt 与 setup.py区别

  1. requirements.txt:

    • 作用: requirements.txt文件通常用于列出项目所依赖的第三方库及其版本信息。

    • 格式
      每行一个依赖项，通常包括库的名称和版本号，例如：

    • 安装依赖: 通过运行pip install -r requirements.txt命令，可以一次性安装所有列出的依赖项。

  2. setup.py:

    • 我们常见的运行命令，“pip install 模块名”或者“python setup.py install”，前者是在线安装，会安装该包的相关依赖包；后者是下载源码包然后在本地安装，不会安装该包的相关依赖包。

    • 在编写相关系统时，如何实现连同依赖包一起打包发布？

    假如我在本机开发一个程序，需要用到python的redis、mysql模块以及自己编写的redis_run.py模块。我怎么实现在服务器上去发布该系统，如何实现依赖模块和自己编写的模块redis_run.py一起打包，实现一键安装呢？同时将自己编写的redis_run.py模块以exe文件格式安装到python的全局执行路径~/Python27/Scripts下呢？

    • 作用: setup.py文件是一个 Python 脚本，用于定义项目的元数据和其他相关配置，例如项目名称、版本、作者信息、依赖关系等。

    • 如何编写

    • 安装依赖: 通过在install_requires参数中指定依赖项，可以告诉 setuptools 安装项目所需的依赖项。当用户运行python setup.py install时，setuptools会自动处理依赖项的安装。

2023年12月22日 python的一些实用的库

1、 库

1、shutil库

shutil.copy()

用于将文件内容从一个位置复制到另一个位置。

示例：shutil.copy('source.txt', 'destination.txt')

将 source.txt 文件的内容复制到 destination.txt 中。

shutil.move()

用于移动文件或重命名文件。

示例：shutil.move('old_file.txt', 'new_directory/new_file.txt')

将 old_file.txt 移动到 new_directory 中，并重命名为 new_file.txt。

shutil.rmtree()

用于删除文件夹及其内容。 示例：shutil.rmtree('folder_to_delete')

递归删除 folder_to_delete 文件夹及其所有内容。

shutil.make_archive()

用于创建归档文件（例如 ZIP 文件）。

示例：shutil.make_archive('archive_name', 'zip', 'folder_to_archive')

将 folder_to_archive 文件夹打包为名为 archive_name.zip 的ZIP归档文件。

shutil.unpack_archive()

用于解压缩归档文件。

示例：shutil.unpack_archive('archive_name.zip', 'extracted_folder') 将 archive_name.zip 解压缩到 extracted_folder。

shutil.disk_usage()

用于获取磁盘使用情况。

示例：usage = shutil.disk_usage('/') 返回根目录 / 的磁盘使用情况，包括总空间、已用空间和可用空间。

2、os库

`os.path` 模块提供的一个函数，用于展开用户路径中的波浪号 (`~`) 或类似的表示家目录的符号。

​ 对于路径 /path/to/some/file.txt，os.path.basename() 将返回 file.txt。

​ 对于文件名 example.txt， 将返回一个元组 ('example', '.txt')，其中 'example' 是文件名部分，.txt 是扩展名部分。

3、imutils

imutils.paths.list_images()

imutils.paths 列出指定目录中的图像文件路径。

imutils.resize()

用于调整图像大小的函数。可以使用 imutils.resize(image, width=new_width) 来调整图像的宽度为指定的 new_width，同时保持纵横比。

imutils.rotate()

用于旋转图像的函数。imutils.rotate(image, angle=degree) 可以将图像按照给定的角度 degree 进行旋转。

imutils.opencv2matplotlib()

用于将 OpenCV 格式的图像转换为 Matplotlib 可以显示的格式。

当涉及到图像处理和计算机视觉任务时，imutils 库提供了一系列实用的功能函数。以下是一些其他常用的 imutils 函数：

imutils.resize()

用于调整图像大小的函数。可以使用 imutils.resize(image, width=new_width) 来调整图像的宽度为指定的 new_width，同时保持纵横比。

imutils.rotate_bound()

与 imutils.rotate() 类似，但是 imutils.rotate_bound(image, angle=degree) 会自动调整图像的大小，确保旋转后的图像完整显示在新的画布内。

imutils.auto_canny()

自适应边缘检测算法，imutils.auto_canny(image, sigma=0.33) 可以根据图像的梯度值智能地计算边缘检测的低阈值和高阈值。

imutils.grab_contours()

用于从不同版本的 OpenCV 中提取轮廓的兼容函数。可以使用 contours = imutils.grab_contours(cnts) 从轮廓对象 cnts 中提取实际的轮廓。

imutils.increase_brightness()

用于增加图像亮度的函数。imutils.increase_brightness(image, value=25) 可以通过添加 value 值来提高图像的亮度。

imutils.translate()

用于平移图像的函数。imutils.translate(image, x=shift_x, y=shift_y) 可以将图像沿着 x 和 y 轴平移指定的偏移量。

4、uuid

​ uuid库中的uuid1()函数用于生成基于主机ID和当前时间的UUID。它基于主机的MAC地址和当前时间戳来生成UUID，因此它是基于时间的UUID。这个函数返回的UUID是32个字符的字符串，表示为32个长度的十六进制数字，通常显示为8-4-4-4-12的分段形式。

uuid4()函数用于生成随机UUID。它使用伪随机数来生成UUID，所以它是不可预测的。返回的UUID也是一个32个字符的字符串，通常显示为8-4-4-4-12的分段形式。

UUID()函数用于将字符串形式的UUID转换为UUID对象。这对于将来自外部来源的UUID字符串转换为Python中的UUID对象很有用。
