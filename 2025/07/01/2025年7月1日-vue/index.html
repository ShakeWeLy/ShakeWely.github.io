computed 是什么？

computed 是 Vue 中的计算属性，它的作用是根据其他数据自动计算出一个新的值。

简单来说，computed 就像一个自动计算的公式，当依赖的数据变化时，它会自动重新计算。

什么是模板 (Template)？

模板就是HTML 结构，它定义了网页的外观和布局。简单来说，模板就是告诉浏览器"这个页面应该长什么样子"。

语法分析

让我逐部分解释：

1. const goToLogin

  • const：声明一个常量

  • goToLogin：函数名

  • =：赋值符号（把函数赋值给变量）

2. () =>

  • ()：函数参数（这里是空的，表示没有参数）

  • =>：箭头函数语法

3. { }

  • 函数体，包含要执行的代码

响应式引用类型

=>

1. w => w.id === id 语法

这是 箭头函数 作为 回调函数 的用法。

语法分析typescriptApply to WorkShopView…const found = workshops.value.find(w => w.id === id);

让我逐部分解释：

workshops.value.find()

  • find() 是数组方法

  • 用于在数组中查找符合条件的第一个元素

  • 返回找到的元素，如果没找到返回 undefined

w => w.id === id

  • w：参数名，代表数组中的每个元素

  • =>：箭头函数语法

  • w.id === id：返回条件表达式的结果（true 或 false）

2. watch 语法

watch 是 Vue 的 监听器，用于监听数据变化并执行相应操作。

语法分析typescriptApply to WorkShopView…watch(() => route.params.id, () => { setWorkshopByRoute();});

让我逐部分解释：

watch(源, 回调函数)

  • 第一个参数：要监听的数据源

  • 第二个参数：当数据变化时要执行的函数

() => route.params.id

  • 这是一个 getter 函数

  • 返回要监听的数据

  • 当 route.params.id 变化时，会触发监听器

() => { setWorkshopByRoute(); }

  • 这是 回调函数

  • 当监听的数据变化时执行

  • 调用 setWorkshopByRoute() 函数

  • 类型：浏览器本地存储

  • 生命周期：永久保存，除非手动删除
