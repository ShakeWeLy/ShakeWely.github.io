1. 自监督与弱监督

自监督：
👉 监督信号从数据本身自动构造，不需要人工标签

弱监督：
👉 有标签，但标签不精确 / 不完整 / 有噪声

常见弱监督形式

1️⃣ 标签不精确（最常见）
  • 只有图像级标签，没有 bbox / mask
  • 如：这张图“有猫”，但不知道猫在哪

📌 弱监督目标检测、弱监督分割

2️⃣ 标签不完整
  • 只标一部分样本
  • 如：数据集中只有 10% 有标注
3️⃣ 标签有噪声
  • 自动生成标签
  • 爬虫数据、规则生成标签

📌 工业场景、日志规则系统（你现在做的 SQL 规则也很像）

典型方法

方法 说明
MIL Multiple Instance Learning
CAM Class Activation Map
伪标签 模型预测 → 当新标签
规则 + 学习 人工规则生成弱标签

1.1 自监督 ≠ 传统无监督

自监督 = 用无标签数据 + 人为设计的监督目标
无监督 = 没有监督目标，主要做结构发现

👉 自监督是“披着无监督外衣的监督学习”

关键差别在「有没有监督信号」

方式 有无标签 有无明确目标
无监督 ❌ ❌
自监督 ❌ ✅（伪标签）
有监督 ✅ ✅

举个直观例子（视觉）

  • 无监督：
    • K-means 聚类
    • PCA / ICA
    • 只是“发现结构”
  • 自监督：
    • 预测旋转角度
    • 图像遮挡 → 复原
    • 对比学习（谁和谁是同一张）
